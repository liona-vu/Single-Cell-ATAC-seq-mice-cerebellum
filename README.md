The project investigates the chromatin accessibility of the developing mice cerebellum using single-cell-ATAQ-seq analysis.

The dataset is based on the published paper below:
Integrated single-cell transcriptomic and epigenetic study of cell state transition and lineage commitment in embryonic mouse cerebellum by Khouri-Farah et al., 2022

Raw scATAC-seq fragments files from E12 - E14 and the metadata can be downloaded using the following links:
https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE178546
https://zenodo.org/records/6097549#.Ygw5My-B30o
